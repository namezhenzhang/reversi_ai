- 进一步探索：
    - 将棋局按阶段分类
    - 选取更优质棋局（胜者局，这点在现有模型中存在）

## src1
基于词嵌入和attention,2head,2layer

- 打分：将下一步的所有可能分别计算分数，然后用mse或crossentropy计算loss
- 分类：将当前棋盘输入，输出为64维对应棋谱位置，
    - 只将正确位置置1，其他为零，crossentropy
    - 正确位置置1，可下位置置0.5，mse

## src2
基于词嵌入和attention,3head,3layer

- 和src1类似

## Resnet

基于resnet18

- 分类：将当前棋盘输入，输出为64维对应棋谱位置，
    - 只将正确位置置1，其他为零，crossentropy
    - 正确位置置1，可下位置置0.5，mse
